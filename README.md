# Spotify Song Prediction and Recommendation System

## Overview
This project aims to build a song prediction and recommendation system using machine learning techniques on Spotify data.

## Project Structure
- `data/`: Contains raw and processed data.
- `notebooks/`: Jupyter notebooks for exploratory data analysis.
- `src/`: Source code for data preprocessing and model training.
- `requirements.txt`: List of dependencies.

## How to Run
1. Clone the repository.
2. Install the required packages using:
   ```bash
   pip install -r requirements.txt
